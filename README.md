# 📧 Spam Email Detector (Machine Learning Project)

A simple and powerful **Machine Learning web app** that detects whether an email/message is **Spam or Not Spam** using Natural Language Processing (NLP).

## 🧠 Project Overview

This project uses **Machine Learning (Naive Bayes)** and **Natural Language Processing (NLP)** to classify emails/messages into:

- ✅ Ham (Not Spam)
- 🚨 Spam

The model is trained on labeled SMS/email dataset and then deployed using **Streamlit**.

---

## ⚙️ Features

- Real-time spam detection
- Simple and clean UI using Streamlit
- Text preprocessing with CountVectorizer
- Fast prediction
- Lightweight ML model

---

## 🛠️ Technologies Used
